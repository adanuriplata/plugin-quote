The WooCommerce ‘request for quote’ plugin enables the customer or visitor to get the rate for the products inside their wish list. This plugin allows the site owner to give out discounted, promotional, special customer or regular customer prices to the customers according to the situation or eve. The customers/visitors can also bargain for the appropriate price and the site owner would have the authority to accept or reject the negotiation.

Installation Guide

This is a quick on installation, and upadating. Please read these below steps really carefully it help you alot.

1.1 Plugin Installation Starting

To install WooCommerce Advanced Request a Quote plugin you must have a working and latest version of WordPress and WooCommerce plugin it self already installed. For information on installing the WordPress, please see the WordPress Codex- How to installing WordPress.
 
To installing the WooCommerce and cofigure the shop please visit the following link Insalling WooCommerce
 
 

To install WooCommerce.

 

  1. Login to you WordPress Site.
  2. Go to Plugins > Add New.
  3. Search or "WooCommerce".
  4. Select Insatall now when you see it by automattic.
  5. Select Activate Now and you're ready for the WooCommerce Wizard!

 

WooCommerce Wizard.

When WooCommerce is activated for the first time , please see these WooCommerce setup wizard. Click lets go to get strted. Or select not rigth now if you prefer manually setup your woocomemrce shop.
 

1.2 Plugin Installation

Once your installation of WooCommerce is done kindly follow these simple steps to install WooCommerce Advanced Request a Quote.
 
  1. Installation from WordPress
  2. Navigate to plugins
  3. Click Install Plugin and click add new
  4. Click upload plugin button
  5. Navigate WooCommerce Advanced Request a Quote.zip on your computer and click install now
  6. When installation is done please click activate to activate plugin
 
Installation Using FTP
Follow these steps to install using FTP
 
  1. Unzip WooCommerce Advanced Request a Quote.zip to desired location on your computer. These extracted files will be uploades later via FTP
  2. Connect to your server via FTP (Use Cute ftp or file zilla tool for uploading files)
  3. Navigat to /wp-content/plugings/server directory
  4. Locate the local directory where you have unzipped the WooCommerce Advanced Request a Quote.zip archive
  5. Upload local folder /WooCommerce Advanced Request a Quote to remote /wp-content/plugins/
  6. To activat the newly installed plugin navigate the Plugins and click to Activate button within the WooCommerce Advanced Request a Quote.

1.3 Plugin Updating

Updating from WordPress Admin Panel
Follow all the important steps to update form WordPress Admin Dashboard.
 
  1. Navigate to Pluings
  2. Find the currently install WooCommerce Advanced Request a Quote plugin and click deactivate
  3. After deactivetion click delete to remove the old version
  4. Click install plugins and click add new
  5. Click the upload pluing button
  6. Navigate the new version archive WooCommerce Advanced Request a Quote.zip on your computer and click install now
  7. When installation i s done please click activate to activate plugin
 
Updating via FTP
Follow the steps below to update using FTP
 
  1. Unzip the new version archive WooCommerce Advanced Request a Quote.zip to a desired location on you computer.
  2. These extracted files will be uploaded later via FTP
  3. Connect to your server via FTP (use cuteFTP or Filezilla)
  4. Locate the local directory whre you have unzipped the new version archive WooCommerce Advanced Request a Quote.zip
  5. Upload local folder /WooCommerce Advanced Request a Quote/ to remote / wp-content/plugins/

Plugin Features

Some of plugin main features
  1. Show “add to quote” button on shop page
  2. Show “add to quote” button on product single page
  3. Hide button for some products
  4. Show and hide “add to quote” button only to login and guest users
  5. Hide prices on all product for your site
  6. Customize full style and color for quote list, button and whole setting controls
  7. Set and hide information in quote list like SKU, Image, Price
  8. Hide and show option to display total, and subtotal
  9. Redirection control over all functions
  10. Form to send quote email to admin
  11. Nice email template which display all information about user and quotation
  12. Option to add quotation list at any page
  13. Show tax on quote page list
  14. Full setting option to control quote list
